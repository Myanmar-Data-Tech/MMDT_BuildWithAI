# Instructions

Welcome! This folder contains project code developed by participants during the **BuildWithAI Workshop**.

## 📁 Folder Purpose
This directory is intended to store all participant-created projects, experiments, and code samples from the workshop sessions.

## 🧑‍💻 Guidelines for Participants

- Create a **separate folder** for your project using a clear naming format:
- Keep your project files organized:
- `README.md` – Describe your project, features, and how to run it
- `requirements.txt` or `package.json` – List dependencies (if applicable)
- Source code files in appropriate folders (`src/`, `app/`, etc.)
