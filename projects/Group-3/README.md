# LANPYA AI Study Planner

##  Overview
LANPYA is an AI-powered study planner that generates personalized learning paths based on:

- Career path

- Current level

- Time availability

The system helps solve the problem of scattered learning resources and lack of structured career guidance. The project uses **Google Gemini 2.5 Flash** to help generate structured learning plans, recommended resources, and weekly study schedules.


##  Objective
- Provide structured learning paths
- Reduce confusion and wasted time
- Help users reach career goals faster

##  Features
- Personalized study plan generation
- Career-based learning paths (e.g., DevOps, Software Engineer)
- Beginner to advanced level support
- Time-based planning (weeks/months)

##  How It Works
The system uses a multi-agent workflow:

- **Concierge Agent** → receives user request  
- **Coach Agent** → processes request  
- **Course Finder Agent** → finds courses  
- **Resource Finder Agent** → finds tutorials/blogs  
- **Weekly Planner Agent** → creates study schedule
